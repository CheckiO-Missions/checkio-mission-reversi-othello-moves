Реверсі - це стратегічна настільна гра для двох гравців, що грається на дошці 8 × 8. Вона була винайдена у 1883 році. Отелло, варіант з фіксованою початковою комбінацією на дошці, був запатентований у 1971 році.

Для цілей цієї місії тобі потрібно написати генератор ходів, щоб знайти всі, доступні для Отелло проти Дездемони на даній дошці. Ігрові фішки Отелло (⚫) та Дездемони (⚪) на дошці - {% if is_js %}масиви{% else %}списки{% endif %} кортежів їх x- та y-координат, обидві координати від 0 до 7.

Твоя функція повинна повертати {% if is_js %}масив{% else %}список{% endif %} усіх можливих ходів, доступних Отелло. Ці ходи мають бути представлені у вигляді кортежів (x, y, flips) де x, y - координати ходу, за якими слідує кількість фішок Дездемони, які цей конкретний хід обертає на фішки Отелло.

Щоб зробити очікувану правильну відповідь унікальною, результатний {% if is_js %}масив{% else %}список{% endif %} ходів повинен бути відсортований за спаданням кількості обернутих фішок за хід (flips). Ходи, що обертають однакову кількість фішок, повинні бути відсортовані за зростанням їх x-координати, остаточний порядок встановлюється за зростанням y-координати.
